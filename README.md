Entrenador-Personal




Aplicación web de entrenamiento personal desarrollada con HTML, CSS y Python, pensada para ofrecer una forma práctica y organizada de gestionar rutinas de ejercicio. Permite a los usuarios crear y personalizar sus planes de entrenamiento según sus objetivos, así como registrar cada sesión con información detallada como ejercicios realizados, series, repeticiones y peso.


Además, la aplicación incluye un sistema básico de seguimiento que facilita visualizar el progreso a lo largo del tiempo, ayudando a mantener la constancia y mejorar el rendimiento.
